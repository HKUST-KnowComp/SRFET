# SRFET
Exploiting Semantic Relations for Fine-grained Entity Typing

### Run Stacking
- Unzip srfet-data.zip to /path/to/data/
- In config.py, set 
```
DATA_DIR = '/path/to/data/'
```
- Run
```
python runstacking.py
```

### Requires
Pytorch, Numpy
